SD 抽卡 - AI 提示词生成器

一款基于 AI 的 Stable Diffusion 提示词生成工具，帮助用户快速生成高质量的图像描述咒语。
✨ 主要功能

    一键 AI 抽卡：输入主题或留空随机生成，AI 自动产出分类整理好的提示词
    九宫格卡槽系统：质量词、艺术家、角色、外貌、服装、动作、物品、场景、构图分类管理
    智能解析：粘贴现有提示词，AI 自动分类并翻译
    权重调节：支持 -100 到 +100 的权重范围，精细控制生成效果

🎭 OC 角色系统

    创建并管理原创角色（OC）及多套服装方案
    支持 A 角色本体 + B 角色服装的自由混搭
    文件集管理，导入导出角色数据

🛠️ 高级特性

    可自定义提示词模板，支持预设保存
    多 API 服务商支持（DeepSeek、硅基流动、OpenAI 兼容）
    对话记忆功能，AI 可参考历史生成
    历史记录回溯，支持撤销/重做
    全量配置备份与恢复

💻 技术栈

React 18 + TailwindCSS + Lucide Icons，纯前端单文件应用，无需后端部署。
🚀 使用方式

直接下载 HTML 文件用浏览器打开即可使用，配置 API Key 后开始抽卡！
